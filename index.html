QUERY_INTERFACE

What is IRONHOOD? +

IRONHOOD is a decentralized control layer that lets physical machines run AI agents on local edge hardware, with credentials and state secured on the Robinhood Virtuals chain. No cloud dependency, no central point of failure.

How do I become a node operator? +

Stake $IRONHOOD tokens to register an edge node. Once your hardware passes the handshake protocol, your node joins the network and earns rewards from the 20% Node Rewards pool.

Why on-chain for hardware control? +

Cryptographic signatures replace vulnerable admin databases. Only sovereign operators holding valid keys can command machinery — no password leaks, no server breaches, no remote takeover.

What happens if a node loses connection? +

Physical failsafes trigger. If a node loses blockchain sync, hardwired relays force the machinery into state-safe standby. Software failure never propagates to physical motion.
